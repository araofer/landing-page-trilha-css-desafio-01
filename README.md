Este projeto é a resolução do **Desafio 01** da Trilha de CSS da [DIO (Digital Innovation One)](https://www.dio.me/). O objetivo deste desafio foi construir uma Landing Page completa a partir do zero, aplicando conceitos modernos de estilização e estruturação web.

## 🎯 Objetivo do Projeto

Desenvolver uma página web responsiva e visualmente fiel a um protótipo (esboço) fornecido, utilizando apenas **HTML e CSS puros**, sem o auxílio de bibliotecas externas ou frameworks (como Bootstrap ou Tailwind).

## ✨ Funcionalidades e Efeitos Aplicados

Durante o desenvolvimento deste projeto, foram aplicadas diversas técnicas avançadas de CSS, destacando-se:

* **Efeito de Overlay (Película Escura):** Utilização de `linear-gradient` combinado com `background-image` para escurecer imagens de fundo, garantindo o contraste e a legibilidade do texto superior.
* **Efeito Parallax Simples:** Implementação da propriedade `background-attachment: fixed` para criar a ilusão de profundidade, onde a imagem de fundo permanece estática enquanto o conteúdo rola pela tela.
* **Layout Flexbox:** Uso extensivo de `display: flex` para alinhamento e distribuição perfeita dos elementos em tela (cabeçalho principal, seções de módulos e rodapé).
* **Efeitos de Transição (Hover):** Interações visuais suaves nos botões e nos textos de destaque utilizando a propriedade `transition` e `transform: translateY`.
* **HTML Semântico:** Estruturação correta do conteúdo utilizando tags como `<header>`, `<main>`, `<section>` e `<footer>`, melhorando a acessibilidade e o SEO.

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
